Introduction
This is a specialized manual explaining the basics of using AbyssBox.
On the left you can find tabs explaining different parts of the webside. As of now, this page exists to showcase the various keybinds that can be found in AbyssBox. Later updates will add more articles for the manual.
Keybinds
When AbyssBox has focus (click on its interface), you can use these keyboard shortcuts:
Playback
  • Spacebar: Pause or Resume
  • Shift+Spacebar: Resume from cursor position
  • Ctrl+Spacebar Or Ctrl+P: Start Note Recording
  • F: Move playbar to the start of the song
  • Shift+F: Move playbar to the start of the loop bar
  • H: Move playbar to the selected pattern
  • [: Move playbar left
  • ]: Move playbar right
Actions
  • Z: Undo
  • Y or Shift+Z: Redo
  • X: Cut selected patterns
  • C: Copy selected patterns
  • V: Paste selected patterns
  • Ctrl+Shift+V: Paste selected Numbers
  • D: Duplicate selected patterns
  • N: New unused pattern
  • Shift+C: Copy selected Instrument
  • Shift+V: Paste selected Instrument

  • A: Select All
  • Shift+A: Select Channel
  • Shift+Click+Drag: Select bars/patterns
  • Esc: Deselect bars/patterns
  • Shift+Up/Down/Left/Right: Move selection bounds

  • Enter: Add selected number of Bars
  • Shift+Enter: Add selected number of Bars behind selection
  • Ctrl+Enter: Add selected number of Channels
  • Backspace: Remove selected number of Bars
  • Ctrl+Backspace: Remove selected number of Channels
  • Ctrl+Up: Move selected channels up
  • Ctrl+Down: Move selected channels down
  • 1/2/3/4/5/6/7/8/9/0: Set selected pattern number
  • Delete: Set selected pattern number to 0

  • M or Shift+S: Mute selected Channels
  • Shift+M: Mute all Channels
  • Alt+M: Invert selected/all muted channels
  • S: Solo selected Channels

  • =: Move selected notes up a semitone
  • Shift+=: Move selected notes up an octave
  • -: Move selected notes down a semitone
  • Shift+-: Move selected notes down an octave

  • B: Loop only selected patterns
  • Shift+~: New song
  • Shift+P: Enter song player
  • R: Random Preset
  • Shift+R: Random Generated Instrument
Prompts
    File:
  • Ctrl+O: Open Song Import Prompt
  • Ctrl+S: Open Song Export Prompt
  • Ctrl+Shift+S: Quick Export Song
  • Shift+D: Open Song Details Prompt
  • ~: Open Song Recovery Prompt

    • Edit:
  • W: Open Move all notes sideways Prompt
  • E: Open Generate Euclidean Rhythm Prompt
  • Shift+E: Open EQ Filter Editor Prompt
  • Shift+N: Open Note Filter Editor Prompt
  • Shift+B: Open Change Beats per Bar Prompt
  • L: Open Change Song Length Prompt
  • Q: Open Channel Settings Prompt
  • Shift+L: Open Limiter Settings Prompt
  • Shift+Q: Open Custom Samples Prompt
Extras
  • Ctrl+Alt+Shift+J: Change all preferences to what Jummbus would prefer.
  • Shift+I: Copy Instrument Code.
  • Caps Lock: Enable midi keyboard inputs.
